# WebFOCUS Language Definition

This project aims to define the WebFOCUS programming language syntax and functionality in a formal way.

## Structure

- `CONCEPT.md`: High-Level Architecture and Use Cases.
- `DESIGN.md`: Detailed design and tech stack.
- `ROADMAP.md`: Planned and accomplished steps.
- `TECHNICAL_DEBTS.md`: Log of technical debts.
- `/src/`: Source code.
- `/test/`: Test cases and tools.
- `/docs/`: Documentation.
